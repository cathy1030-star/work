# 📅 任務管理中心 (Daily Task Manager)

一個基於 Web 技術開發的輕量化、活潑且功能強大的個人任務管理工具。專為提升日常工作效率而設計，支援複雜的週期性任務管理與多媒體備註。

### 🌟 主要功能
- **任務管理**：支援新增、修改、刪除及「已完成」狀態切換。
- **獨立週期性實例 (Instance Overrides)**：支援每天、每週、每月重複（限制預覽未來 3 個月）。每個日期實例均可獨立編輯內容、時間、刪除或標記完成，不影響系列中的其他天數。
- **嚴格雙向同步**：清單檢視與月曆圖表共享過濾邏輯。搜尋關鍵字、日期區間時，兩者內容保持絕對一致。點擊月曆可自動跳轉並定位清單。
- **工作 SOP**：獨立的 SOP 紀錄區塊，支援全螢幕編輯與複製。
- **智慧過濾系統**：支援多重關鍵字即時高亮搜尋、精準日期區間篩選，以及逾期任務自動紅字警示。
- **豐富多媒體**：任務備註與 SOP 支援 Base64 圖片上傳，並支援直接從剪貼簿 (Ctrl+V) 貼上截圖。
- **離線優先與隱私**：所有資料完整儲存於使用者的瀏覽器 LocalStorage，無需伺服器或後端資料庫，保障資料隱私。

### 🛠 技術棧
- **前端框架**: Bootstrap 5.3 (視覺與佈局)
- **日曆組件**: FullCalendar 6.1 (排程管理)
- **圖示系統**: Bootstrap Icons
- **資料持久化**: Browser LocalStorage API

### 🚀 如何使用
1. 將專案下載或 Clone 到本地。
2. 直接使用瀏覽器開啟 `index.html` 即可開始使用。

### 📦 部署 GitHub Pages
1. 將 `index.html` 上傳至您的 GitHub 儲存庫。
2. 在 `Settings > Pages` 中開啟 GitHub Pages 服務，即可獲得專屬的線上任務管理網址。
