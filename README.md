# 📅 任務管理中心 (Daily Task Manager)

一個輕量化、活潑且功能強大的個人任務管理網頁工具。

### 🌟 主要功能
- **任務管理**：支援新增、修改、刪除及「已完成」狀態切換。
- **週期性任務**：支援每天、每週、每月重複，且各日期實例可獨立標記完成或修改備註。
- **多重檢視**：提供「清單檢視」與「月曆圖表 (FullCalendar)」。
- **工作 SOP**：獨立的 SOP 紀錄區塊，支援全螢幕編輯與複製。
- **智慧功能**：支援多重關鍵字高亮搜尋、日期區間過濾、逾期自動紅字提醒。
- **多媒體支援**：支援圖片上傳及剪貼簿直接貼上截圖。
- **離線儲存**：所有資料儲存於瀏覽器 LocalStorage，無需後端資料庫。

### 🚀 如何使用
1. 將專案下載或 Clone 到本地。
2. 直接使用瀏覽器開啟 `index.html` 即可開始使用。
